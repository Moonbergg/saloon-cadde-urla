# Saloon Cadde — Urla

Urla / İzmir'de gelin saçı & makyajı, saç bakımı, tırnak ve güzellik hizmetleri sunan
**Saloon Cadde** için premium, dönüşüm odaklı landing demo. WhatsApp randevu odaklıdır.

Tek dosyalık statik site (`index.html`) — build/derleme gerektirmez.

**Canlı:** https://moonbergg.github.io/saloon-cadde-urla/

## İletişim bilgilerini güncelleme
`index.html` sonundaki `CONFIG` bloğunu düzenleyin (WhatsApp, telefon, Instagram).
Tüm butonlar ve linkler bu değerlerden otomatik üretilir.

## Yerelde görüntüleme
```bash
python -m http.server 5500
# http://localhost:5500
```
